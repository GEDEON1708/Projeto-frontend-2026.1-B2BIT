# Mini Twitter

Este workspace tem duas pastas:

- `mini-twitter-backend-main`: backend usado no desafio
- `mini-twitter-frontend`: frontend da aplicacao

## Rodando o projeto

### Backend

```bash
cd mini-twitter-backend-main
bun install
bun run seed
bun run dev
```

### Frontend

Em outro terminal:

```bash
cd mini-twitter-frontend
npm install
npm run dev
```

## Enderecos

- API: `http://localhost:3000`
- Frontend: `http://localhost:5173`
